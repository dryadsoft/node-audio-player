# node-audio-player

## server 로컬 실행

> express 버전

```s
$ cd server && yarn dev:watch
```

> nest.js 버전

```s
$ cd server-nestjs && npm run start:dev
```

## client 로컬 실행

```s
$ cd client && npm start
```

## 음악폴더

> express 버전

- server/music/songs

> nest.js 버전

- server-nestjs/music/songs

## server build

> express 버전

```s
$ cd server && yarn build
```

> nest.js 버전

```s
$ cd server-nestjs && npm run build
```

## client build

```s
$ cd client && npm run build
```
